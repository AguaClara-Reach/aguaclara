Pipeline¶

This utility is used to provide useful functions for all things related to pipeline design.

aguaclara.core.pipeline.flow_pipeline(diameters, lengths, k_minors, target_headloss, nu=<Quantity(1e-06, 'meter ** 2 / second')>, pipe_rough=<Quantity(0.12, 'millimeter')>)[source]¶

This function takes a single pipeline with multiple sections, each potentially with different diameters, lengths and minor loss coefficients and determines the flow rate for a given headloss.

Parameters:
  • diameters (numpy.ndarray) – list of diameters, where the i_th diameter corresponds to the i_th pipe section
  • lengths (numpy.ndarray) – list of diameters, where the i_th diameter corresponds to the i_th pipe section
  • k_minors (numpy.ndarray) – list of diameters, where the i_th diameter corresponds to the i_th pipe section
  • target_headloss (float) – a single headloss describing the total headloss through the system
  • nu (float) – The fluid dynamic viscosity of the fluid. Defaults to water at room temperature (1 * 10**-6 * m**2/s)
  • pipe_rough (float) – The pipe roughness. Defaults to PVC roughness.
Returns:

the total flow through the system
Return type:

float
